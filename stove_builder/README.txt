В процессе верстки данного макета, были решены следующие задачи:
сверстать с использованием техник отзывчивого дизайна для разрешений экрана 320-1160 пикселей;
динамически создавать и удалять содержание пункта меню «ЕЩЕ» появляющегося при ширине видимой области браузера пользователя меньше 1150 пикселей;
создать выпадающее меню, появляющееся  при ширине видимой области браузера пользователя меньше 600 пикселей;
сверстать форму заказа звонка.
За динамическое создание пунктов меню, отвечает скрипт «More submenu dynamically creation.js».
Скрипт работает в два этапа, инициализация страницы, и изменение размеров.
Форма заказа звонка вызывается и скрывается скриптом «poopUp_handler.js».
Форму решил позиционировать без плагинов.
За центрирование выпадающих пунктов меню, относительно родительских пунктов отвечает скрипт «submenu position.js».
